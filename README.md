# Grading Program

A simple Java Swing desktop application for managing students' grades.

## Overview
This project allows the user to:
- Select a doctor
- Select a course
- Add a student name and grade
- Search for a student's grade
- View all saved grades
- Calculate the average, minimum, and maximum grade

The application stores data in text files based on the selected doctor and subject.

## Technologies Used
- Java
- Java Swing
- File Handling

## Features
- Simple desktop GUI
- Store grades in text files
- Search for student grades
- Display all students and grades
- Show average, highest, and lowest marks

## Project Structure
- `Main.java` → Starts the application
- `Frame1.java` → First screen for doctor selection
- `Frame2.java` → Main grading screen
- Image files → Used in the interface

## Notes
- The project currently uses hardcoded local file paths.
- You may need to update file paths on your device before running it.
- Make sure the required text files and images are available in the correct locations.

## How to Run
1. Open the project in IntelliJ IDEA or any Java IDE.
2. Update the file paths inside the code if needed.
3. Run `Main.java`.
